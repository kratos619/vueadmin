# Vue-Admin

Simple Dashboard That Managed Create , Read, Update, Delete , In Real Time Using Laravel And Vue JS On Single Page

## Features

- User Management
- User Autherization and Authentication
- RestFull Api's
- Api Protection (JWT)
- Security
- Single Page Application
